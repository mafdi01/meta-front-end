# Requirements / User Stories

- You have to choose one client persona to develop a home page for. 
- Develop a home page with a header, main content and footer:
  - Your header should contain the client logo at the top of the page with a navigation menu underneath.
    - The nav links: 
      1. Homepage
      2. Products page
      3. Contact form
      4. Company Information page
  - The main content should be a large promotional banner with three columns with text and images below it. 
    - The main content should feature a large banner (full width column) that links to the client's current promotional offer.
    - Below the promotional banner There are three columns. Each column should contain an image followed by a heading and some text. These can contain offerings or information about the client.
    - To simplify the main content Consider splitting the content into two rows.
  - Your footer will have two columns: the first column should have a small logo and the second one should have copywriting information.
- When it comes to styling, you'll have to create the appropriate look and feel for your client's target audience. Remember, you want to create intentional engagement!
- The module covers the most essential features of your project. This includes: selecting your fictional client, working out your layout, positioning the elements, planning the user experience, creating themes and setting up your local development environment.
- The home page needs to meet certain criteria regarding the following:
  - Visual layout
  - Semantic structure
  - CSS layout
  - CSS styling
  - CSS effects
